# Lodes LLC Premium Website

A Vite + React one-page website for Lodes LLC.

## Deploy on Vercel

1. Upload these files to the GitHub repository connected to Vercel.
2. Commit changes to the `main` branch.
3. Vercel will automatically redeploy.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
